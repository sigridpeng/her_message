# 她留下的訊息 (Her Message)

這是一款**溫馨劇情的點擊解謎遊戲**。玩家將扮演主角，在前往女友住處卻發現人不見了的情況下，透過探索房間內的物品與線索，拼湊出她留下的訊息。

## 遊戲特色
- **遊戲類型**：點擊探索 / 敘事解謎 (Point-and-Click Puzzle)
- **難度等級**：入門級 (Entry-level) - 適合所有人體驗
- **核心體驗**：在寧靜的房間內尋找線索，體驗一段感人的溫馨故事。

## 運行建議
- **裝置建議**：建議使用**電腦 (PC/Mac)** 開啟以獲得最佳的操作體驗。

## 線上遊玩
您可以直接在瀏覽器中開啟遊戲：
[https://sigridpeng.github.io/her_message/](https://sigridpeng.github.io/her_message/)

## 本地運行 (Run Locally)

**前置需求：** Node.js

1. 安裝套件：
   `npm install`
2. 啟動開發伺服器：
   `npm run dev`
3. 在瀏覽器打開 `http://localhost:3000` (或控制台顯示的網址)

## 技術棧
- React + Vite
- Tailwind CSS
- TypeScript
